# AI-Powered Operational Risk Insights Assistant

## Problem Statement
Operational teams handle large volumes of issue data, making it difficult to identify recurring risks and patterns manually.

## Solution
This project uses AI and NLP techniques to:
- Analyze customer issue data
- Detect sentiment and risk trends
- Cluster similar issues automatically
- Generate AI-driven insights via a conversational assistant

## Tech Stack
- Python
- Pandas
- Scikit-learn
- NLP (TF-IDF, clustering)
- Transformers (FLAN-T5)
- Google Colab

## Key Features
- NLP-based issue clustering
- Sentiment-driven risk identification
- AI-generated insights
- Interactive AI assistant for business questions

## Business Value
Transforms raw issue data into actionable insights to support operational decision-making and process improvement.
